## Description of Scripts

### Script 1: System Identity

This script shows system information. It helps in understanding where **Git** runs.

### Script 2: Package Inspector

This script checks if **Git** is installed. It also shows the version and installation path of **Git**.

### Script 3: Disk Auditor

This script reviews disk usage. It lists files in the directory.

### Script 4: Log Analyzer

This script reads system logs. It counts occurrences of keywords like "error."

### Script 5: Manifest Generator

This script collects user input. It generates a text file with project information.

## How to Run the Project

1. Open terminal (Ubuntu/WSL)

2. Go to the scripts folder:

```

cd oss-audit-git/scripts

```

3. Run scripts using:

```

python3 script_name.py

```

Example:

```

python3 script1_system_identity.py

```
